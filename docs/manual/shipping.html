Shipping

Group the project's panels into crates and print a packing list for each one so the truck loads cleanly and nothing gets left behind. This tab is where you create and manage the crates themselves; you decide which panel goes in which crate over on the Tracking tab.

Two tabs, two jobs

Crates are built and printed here on Shipping, but panels are assigned to crates on the Tracking tab using the Crate # column (or its bulk-assign tool). Set up your crates here first, then go assign panels there.

What this tab is for

A finished job ships as a set of crates. The Shipping tab is your crate list: you create a crate, give it a number and a short description, and the panels that were assigned to it on Tracking show up inside it automatically. When a crate is ready, you print its packing list — the sheet that rides with the crate and tells the receiver exactly what's inside.

How to crate and ship a job

  1. Set up the crates here. On this Shipping tab, add a crate for each box, bundle, or load. Give each one a clear Crate # and a description so the floor knows what it's for.
  2. Assign the panels on Tracking. Go to the Tracking tab and put each panel into its crate using the Crate # column (or the bulk-assign tool for many panels at once). The panels then appear inside the matching crate here.
  3. Print the packing lists. Come back to Shipping and click Print Packing List on each crate. Hand the lists to the loading crew.
Save the project first

Make sure your latest changes have saved before you print a packing list — watch the save indicator near the top-right of the project header. Printing pulls from the saved project, so a list printed before saving may be missing a panel or a crate change.

What a crate card shows

Each crate is an expandable card. Click it to open it and see everything assigned to that crate:

Crate #
The crate's number. Editable — type the number you want stenciled on the box.
Description
A short, editable label for the crate, e.g. “Wing A – East Side.” Helps the crew and the receiver tell crates apart.
Panel list
Every panel assigned to this crate over on Tracking, showing its type, dimensions, color, and quantity. Each panel has a remove button if it landed in the wrong crate.
Print Packing List
Prints the packing list for just this crate. See Printing & Labels.

Managing your crates

Add a crate
Creates a new, empty crate ready to be numbered and described. Add one per box or load.
Delete a crate
Removes the crate. The panels that were in it simply become unassigned again on Tracking — they aren't deleted from the job.
Reorder crates
Drag a crate card up or down to put your crates in load order or any order that makes sense.
Remove a panel
Use the remove button next to a panel inside a crate to take just that panel out, without touching the rest of the crate.

“List as quantities”

At the top of the tab is a project-wide List as quantities checkbox. It changes how repeated panels are shown:

Off (default)
Every panel is listed on its own line, one row per piece.
On
Identical panels collapse into a single “type × count” row instead of repeating. The setting applies to both the on-screen view and the printed packing list.
When to use the quantities view

Turn List as quantities on for large, repetitive jobs — when a crate holds dozens of the same panel, a single “type × count” line is far easier to read and count against than dozens of identical rows. For mixed, one-off crates, leave it off so every piece is spelled out.

Phased jobs

If the project uses phases, the Shipping tab only shows the crates for the castings in the currently active phase. Switch phases to see and print the crates for another phase.
