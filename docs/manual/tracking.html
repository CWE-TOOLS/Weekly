Tracking

The shop's live status board for the job. Tracking follows every panel as it moves through the production phases — from the mill to the loading dock — and is where you assign panels to shipping crates. If you want to know what's done and what's left, this is the tab to open.

Where it lives

It's the Tracking tab in a project's left sidebar — the fourth tab, right after Casting Inventory.

What this tab is for

Tracking is the day-to-day record of where each panel is. Every panel in the job gets a row, and every production phase gets a column. As a panel finishes a stage, you click that stage's cell to mark it done — a checkmark appears. Anyone looking at the board can see, at a glance, exactly how far along each panel is. Tracking is also where you sort panels into shipping crates and flag any panel that gets rejected.

You need a saved project with at least one casting first

Tracking is built from your castings. Save the project and add at least one casting in Casting Inventory before this tab will have anything to show.

Tracking builds itself from Casting Inventory

This is the most important thing to know about this tab: you do not add or remove panels here. The Tracking sheet is generated automatically from Casting Inventory. Every panel you enter there shows up here, ready to track.

To change the panels, go to Casting Inventory

If a panel is missing, has the wrong dimensions, or the count is off, fix it in Casting Inventory. Tracking will update itself to match. Don't go looking for an “add panel” button here — there isn't one, by design.

How the board is laid out

The board is organized into one expandable section per casting. Click a casting's heading to open or close it. Inside each section is a table that lists all of that casting's panels, one per row, with these columns:

Panel ID
Generated automatically for each panel — you don't type it.
Type / dimensions
What the panel is and its size, carried over from Casting Inventory.
Color / sealer
The color and sealer called for on that panel.
Quantity
How many of that panel there are.
Crate #
A selector to assign the panel to a shipping crate. See Assigning panels to crates below.
Phase columns
One column for each production phase. Click a cell to mark that phase complete for that panel.

The production phases

Each panel moves through the same eleven stages, left to right. Click a panel's cell under a stage to mark it complete; click again to clear it. Here's what each column means:

ColumnStageWhat it means
MILLMillThe mill / shop work to build the form.
FOForm OutThe form is laid out and ready.
CASTCastThe panel has been cast (concrete poured).
DEMOLDDemoldThe panel has been removed from the form.
FINISHFinishSurface finishing work is done.
SEALSealThe panel has been sealed.
STRIPSStrip formsThe forms have been stripped.
DRILLDrillingAny required holes have been drilled.
CRATECratingThe panel has been crated for shipment.
FINALFinal inspectionThe panel has passed final inspection.
LOADLoadingThe panel has been loaded onto the truck.

How to update tracking

  1. Open the casting. Click the heading of the casting whose panels you're updating to expand its section.
  2. Find the panel. Locate the panel's row by its Panel ID, type, or dimensions.
  3. Mark the stage done. Click the cell where the panel's row meets the finished phase's column. A checkmark appears to show that stage is complete. Click again to undo it.
  4. Move along as work happens. Keep checking off stages left to right as each panel progresses through the shop. The board always reflects the latest status.
Everything saves itself

Like the rest of the portal, Tracking saves automatically a moment after you click. There's no Save button — just watch the save indicator near the top-right of the project header.

Assigning panels to crates

Each panel row has a Crate # selector. Use it to say which shipping crate the panel goes in. The crate choices in that selector come from the crates you set up on the Shipping tab — so build your crates there first, then come back here to assign panels to them.

Bulk assign several panels at once

Assigning crates one row at a time is slow on a big job, so you can do it in bulk. Select a range of panels — or several panels at once — and a bulk toolbar appears. Pick a crate in that toolbar and every selected panel is assigned to it in a single step.

  1. Select the panels. Choose a range of rows, or pick several panels you want in the same crate.
  2. Use the toolbar that appears. A bulk toolbar shows up once panels are selected.
  3. Assign the crate. Choose the crate in the toolbar; all the selected panels are assigned to it at once.
Crates live on the Shipping tab

You create and name crates on Shipping, then assign panels to them here. The Shipping tab is also where packing lists are printed.

Focusing the view with the phase picker

At the top of the board is a chip bar that lets you toggle which phase columns are showing. By default every phase is visible. Turn off the stages you don't care about right now to narrow the board down to just the columns you're watching — handy when, say, you only want to see what's left to SEAL and CRATE. This only changes what you see on screen; it doesn't affect any panel's status.

Narrow the board to what matters today

On a wide board, hiding the phases that are already done (or not yet relevant) makes it much easier to scan the few columns the shop is actively working through.

Rejecting a panel

If a panel doesn't pass, you can mark it as Rejected. Rejected counts flow back to Casting Inventory, where they feed remake castings — the system can spin up replacement panels to recast. This keeps the job's true panel count honest: a rejected panel still needs to be made, and the remake mechanism makes sure it isn't forgotten. Full details on remakes are on the Casting Inventory page.

Track phases (phased projects only)

If the project uses Phases, Tracking gains a Track phases chip bar. Use it to choose which of the project's phases appear on the sheet, so you can focus the board on one wing or delivery wave at a time. This chip bar only shows up on projects that have phases turned on — see Phases.

Printing tracking sheets

The Print Tracking button opens a dialog where you pick which castings' tracking sheets you want to print — choose one, several, or all of them. See Printing & Labels for the full rundown on printing from the portal.
