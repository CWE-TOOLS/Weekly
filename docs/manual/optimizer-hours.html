Optimizer Hours

Estimate the labor manhours for each casting, task by task. You build a shared list of tasks once, then enter the hours for each casting. These estimates feed the schedule — they help set the planned start and ship dates you see in the project list.

Set up the project first

Save the project, then add at least one casting in Casting Inventory. Optimizer Hours has nothing to fill in until the project has castings.

What this tab is for

Optimizer Hours is where you answer the question “how many manhours will each casting take?” You break the work into tasks (also called phases) — milling, casting, finishing, and so on — and put an hours estimate on each one for each casting. The tasks are shared across the project, so they stay consistent from casting to casting, while the hours are entered separately per casting.

These numbers don't just sit here. They roll up into the labor estimate that helps plan when the job runs, which is part of what produces the planned start and ship dates shown in the project list. Once the job is built, you can compare these estimates against what was actually worked on the Actual Labor tab.

The Notes box

At the top of the tab is a project-level Notes box. Use it for anything about the hours or how the job is being optimized — assumptions you made, crews you're planning around, or reminders for whoever reviews the estimate. It applies to the whole project, not to one casting.

How the workspace is laid out

The project's castings appear as pills across the top. Below them, the workspace shows three castings side by side — a left, center, and right column — so you can compare and copy hours between them. To bring other castings into view, click their pills or use the prev/next arrows to scroll the columns along.

Each column is one casting. Inside a column are the task cards for that casting, stacked top to bottom, with a Grand Total at the bottom.

Phased projects

If the project uses Phases, only the castings in the phase you're viewing appear as pills here. Switch phases to estimate the rest.

What's on a task card

Task / phase name
What the task is, e.g. “Mill,” “Cast,” “Finish.” Editable — click to rename it.
Hours
The estimated manhours for this task on this casting. Editable — click and type a number.
Description
A collapsible note about the task. Expand it to add detail about what the hours cover; collapse it to keep the column tidy.

Each card also lets you copy its hours so you can paste them onto the matching task in another casting, and delete the task if it doesn't apply.

The Grand Total card

At the bottom of each column, the Grand Total card adds up the hours from every task on that casting. It updates as you type, so it always reflects the total estimated manhours for that casting.

Estimating hours for a casting

  1. Pick the casting. Click its pill so it appears in one of the three columns.
  2. Add the tasks. Use that column's + Add Task button to add each task or phase the casting needs.
  3. Name each task. Click the task name and type what it is (Mill, Cast, Finish, and so on).
  4. Enter the hours. Click the hours field on each card and type your manhour estimate.
  5. Add detail if it helps. Expand the description on a card to note what the estimate assumes.
  6. Check the total. Read the Grand Total at the bottom to confirm the casting's total looks right.
Reuse hours across similar castings

When two castings are alike, you don't have to retype everything. Bring both into view, copy the hours from a task on the first casting and paste them onto the matching task on the second. It's the fastest way to estimate a run of near-identical pieces, then tweak the few cards that differ.

Printing the estimate

The Print Optimizer Hours button prints the estimate with one casting per page, so each casting's task list and total comes out on its own sheet. See Printing & Labels for more on printouts.

Saving

The portal saves your typing automatically as you go — watch the save indicator near the top-right of the project header. There's no separate “Save” button to remember.
