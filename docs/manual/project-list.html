The Project List

The front door of the Project Portal: a searchable, sortable table of every job in the shop. This is where you find an existing project or start a new one.

Getting here

Open the Weekly Schedule and click the Project Portal button in the top navigation. You'll land right on the project list.

What this view is for

The project list is the portal's default view — every job in one table. Use it to spot your own jobs, check where each one stands, jump into a project to do work, or create a brand-new project. Each row is one job; click it to open that project.

How to find a job

  1. Open the portal. From the Weekly Schedule, click Project Portal.
  2. Search or sort. Type in the Search by project # or name… box to filter the list as you type, or click a column header to sort by it.
  3. Open it. Click anywhere on the row. The project opens on its Project Info tab.
Search matches both

The search box filters on both the project number and the project name at the same time, so you can type either one — or part of one — to narrow the list instantly.

Sorting the table

Click any column header to sort the whole list by that column. Click the same header again to reverse the order (for example, newest dates first, then oldest first). This works on every column — sort by Status to group jobs by stage, by PM to gather your own jobs, or by a date column to see what's coming up.

The columns, in order

ColumnWhat it shows

Project #The unique number for the job — the same number used everywhere in the portal.
Project NameThe customer-facing name of the job.
PMThe project manager assigned to the job, so you can spot your own work at a glance.
Kickoff DateWhen the job was kicked off.

Optimizer Start DateWhen the job is planned to start, from the scheduling side of the system. Read-only here.

Planning dates are read-only

Optimizer Start Date and Next Opt. Ship Date come from the scheduling / optimizer side of the system. They appear in the list so you can see when each job is planned to start and next ship, but you don't edit them here.

Starting a new project

The + New Project button at the top-right opens the New Project dialog.

  1. Click + New Project. It's in the top-right corner of the list.
  2. Enter a Project #. Required, and it must be unique — if you type a number that already exists, the portal blocks it and shows an error.
  3. Enter a Project Name (optional). You can add or change this later on Project Info.
  4. Click Create. The new project opens immediately on its Project Info tab, ready to fill in.
Project numbers must be unique

Two projects can't share a number. If the number is already in use, you'll get an error and the project won't be created — pick a different number.

Status and PM are set inside the project

The list shows each job's status and PM, but you don't set them here. Open the project and use the status bar on Project Info to change them.
