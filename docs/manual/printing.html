Printing & Labels

One place to find every printout the portal can produce — what each one is and where its button lives. Each printout opens your browser's print dialog, so you choose the printer and paper there.

How printing works

Every printout below opens your browser's standard print dialog. Pick the right printer and paper size there before you print. To save a printout as a file instead, choose Save as PDF as the printer.

Every printout, at a glance

Project cover sheet
The Print button in the project header while you're on Project Info. Prints the project's cover / info sheet.
Folder labels
The Print Label button in the project header (Project Info). Opens a dialog to pick which castings to print (with Select all). Each label shows the project name and casting # on top, with the color log below — good for labeling job folders.
Tracking sheets
The Print Tracking button on the Tracking tab. Pick which castings to print.
Casting layout
The Print Layout button on the Casting Layout tab. Prints the table-arrangement diagram.
Optimizer hours
The Print Optimizer Hours button on the Optimizer Hours tab. Opens a dialog with Select all / Select none, then prints one casting per letter-sized page.
Color log
The Print Color Log button on the Color Log tab. Prints the full color-log form.
Batch tickets
The Print Batch Tickets button on the Batch Tickets tab. Pick which castings, then prints one page per batch, color-coded by batch type.
