Classroom (CR) Notes

A running record of the three project “classroom” meetings — kickoff, mid-project, and pre-production. Each classroom comes with a standard agenda checklist, a place to capture notes, and a task list, so the same review happens on every job.

Where it lives

The CR Notes tab in a project's sidebar. The heading on the tab reads Classroom Notes.

What “classrooms” are

A classroom is a structured project meeting. Every project goes through three of them, each with its own built-in agenda:

Classroom 1
Standard kickoff checklist — review drawings, classify the project, discuss the sample/mix/finishing notes, confirm castings and layout needs, walk through Optimizer hours, break the job into phases, and more.
Classroom 2
Standard mid-project checklist — review shop and 3D drawings, material usage and specialty POs, phased tickets, the sample color/finish in depth, the draft tracking sheet, casting layout, and the batching tickets.
Classroom 3
Standard pre-production checklist — confirm the shop folder is complete, final drawings and parts lists, casting layout done, all batching tickets accurate, final QA matches tracking, Optimizer hours finalized, and crate sizes/quantities confirmed.

Using the tab

  1. Pick a classroom. Use the three sub-tabs at the top (Classroom 1 / 2 / 3) to switch between meetings. Only one is shown at a time.
  2. Work the agenda. Each classroom shows its numbered agenda — use it as a live checklist during the meeting so nothing gets skipped. Some agenda items are tagged (for example, “Review in Classroom 2” or “Homework”) to flag when they're really handled.
  3. Capture tasks. Under Tasks, click + Add Task to log action items that come out of the meeting for that classroom.
  4. Take notes. Type meeting notes into the notes box for that classroom. Each classroom keeps its own separate notes.

General Notes

Below the classroom panels is a General Notes card. Unlike the per-classroom notes, this one is shared across all three classrooms — use it for project-level notes that apply no matter which meeting you're in.

Why it matters for onboarding

The agendas are the same on every job, so a new estimator or PM can run a classroom meeting just by walking the checklist top to bottom. It's the fastest way to learn what “a complete project” looks like at CWE.

Save the project first

You need a saved project before you can add tasks. If you just created the project, fill in the Project Info and let it save, then come back.
