Color Log

The mix recipe for the project. The Color Log records every ingredient that goes into the concrete — cement, sand, aggregates, pigments, additives, grout, and fill coat — laid out like the printed “Color Log” form the shop has always used. It is the single source the Batch Tickets tab scales from, so getting it right matters.

Where it lives

It's the Color Log tab in a project's left sidebar. On screen it looks like the paper form, with the header Concreteworks East — Rev. 2025 across the top.

What this tab is for

The Color Log is the recipe of record for the job's concrete. Everyone who batches, finishes, or seals a panel reads from it, and the Batch Tickets tab uses it to do its math — it scales every ingredient up or down to match the volume each casting needs. Because of that, the Color Log has to be complete and accurate before batch tickets will work.

Batch Tickets depend on Sand

For Batch Tickets to generate, the recipe must include a Sand entry (and the overall amounts filled in). The batch generator scales the whole recipe based on the sand weight — no sand, no ticket. Keep your sand amounts and units correct.

The toolbar

Across the top of the Color Log are a few controls that set up the form:

Load Standard Color
A dropdown of saved preset recipes. Pick one to drop a known mix into the log as a starting point, then adjust it for this job.
Multiple color logs
A toggle that lets the project hold more than one named color log (for jobs with several mixes). When it's on, a pill strip appears with one pill per log, plus + Add color log and Delete to manage them.
⋯ (more) menu
Opens a small menu. The main item is Save as Standard Color, which saves the current log as a reusable preset — a dialog asks you to name it. After that, the recipe shows up in the Load Standard Color dropdown for any project.
Notes
A project-level notes box for anything about the color log that doesn't fit the form fields.
“Multiple color logs” is one-way

Once you turn the Multiple color logs toggle on, it can't be turned back off for that project. Only switch it on when the job genuinely needs more than one mix.

How to build a color log

  1. Start the recipe. Either pick a preset from Load Standard Color to start from a known mix, or start blank and enter everything by hand.
  2. Fill the header. Enter the Sample # (the color name), Temperature, Date, and Made By. The Project is filled in for you and can't be edited.
  3. Set the cement type. Choose the STANDARD / CUSTOM toggle, then pick the Cement Type — White Portland, Gray Portland, or Other.
  4. Enter base ingredients and additives. Fill the Base Ingredients table and the Additives table. Each row holds an amount, a unit, and a note; use + Add to add more rows. Make sure a Sand entry is included.
  5. Pick the cast method. Choose Cast Method — Spray Up, Direct Cast, or Other. This choice changes how Batch Tickets handles materials, so set it correctly.
  6. Enter the rest of the mix. Fill the Aggregates, Pigments, Grout Type, and Fill Coat tables.
  7. Add notes. Use the Finishing Notes and Sealing Notes boxes for instructions the floor needs.
Saving

The portal saves your typing automatically as you go — watch the save indicator near the top-right of the project header. There's no separate “Save” button to remember.

The form, section by section

The Color Log mirrors the paper form. Here's what each section holds:

SectionWhat it holds
Header fieldsSample # (the color name), Temperature, Date, Made By, and Project (auto-filled and read-only).
STANDARD / CUSTOMA toggle marking whether this is a standard mix or a custom one for the job.
Cement TypeWhite Portland, Gray Portland, or Other.
Base IngredientsThe core mix materials. Each row: ingredient/amount, unit, and a note. + Add adds rows. Include a Sand entry.
AdditivesAdmixtures and additions. Each row: amount, unit, and a note. + Add adds rows.
Cast MethodSpray Up, Direct Cast, or Other. Affects how batch tickets handle materials.
AggregatesThe aggregate materials in the mix, one per row.
PigmentsEach row: pigment, %, weight, and unit.
Grout TypeEach row: name and ratio %.
Fill CoatEach row: fill pigment, %, weight, and unit.
Finishing NotesFree-text instructions for finishing the panels.
Sealing NotesFree-text instructions for sealing the panels.

Printing the Color Log

The Print Color Log button prints the full form, exactly as it appears on screen, for the shop floor or the job folder. See Printing & Labels.

Save before you print

Save the project first so your latest changes are on the printout. The portal saves automatically a moment after you type — wait for the save indicator to read Saved before printing.
