Phases

A way to split a large job into smaller pieces — building wings, delivery waves, or any natural grouping — so several tabs get their own per-phase view instead of mixing everything together in one long list.

One-way switch — there's no going back

Once phases are enabled for a project, they cannot be turned back off. Only enable phases when the job is genuinely phased. If you're not sure, leave it off — a project runs perfectly well without phases.

What this tab is for

Most jobs are simple enough to handle as one big list of castings. But a large job — say a building with three wings, or a project that ships in waves — gets unwieldy when everything is piled together. Phases let you carve the job into named pieces. The tabs that deal with day-to-day production then show one phase at a time, so the screen only shows the castings you're working on right now.

How to enable and use phases

You turn phases on from the Phases section at the bottom of the Project Info tab.

  1. Open Project Info. Scroll to the Phases section at the bottom of the tab.
  2. Turn it on. Click Enable Phases. (Read the warning above first — this can't be undone.)
  3. Add your phases. Create a phase for each piece of the job and give it a clear name (for example “Wing A” or “Delivery 1”).
  4. Reorder and rename as needed. Drag phases into the order that makes sense, and rename them at any time.
  5. Switch between phases. On any phase-aware tab, use the phase tabs at the top of the section, or the phase switcher dropdown in the project header, to choose which phase you're looking at.
Decide on phases early

Set up your phases before loading lots of castings — ideally right after kickoff. Each casting belongs to a phase, so deciding early means every casting lands in the right place from the start instead of being sorted out later.

Which tabs are affected

When phases are on, the production tabs gain per-phase views. Everything else stays project-wide — shared across all phases — so you only set it once for the whole job.

Casting InventoryPer-phase — each casting belongs to a phase.

Choosing which phases to track

On the Tracking tab, a Track phases chip bar lets you pick which phases show up on the tracking sheet. Turn off the phases you're not working on right now to keep the sheet focused on the castings actually moving through the shop.

Where the switch lives

Phases are turned on from Project Info, at the bottom of the tab. Once enabled, the phase switcher appears in the project header so you can change the active phase from anywhere in the job.
