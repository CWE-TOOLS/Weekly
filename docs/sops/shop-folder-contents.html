9.1

Standard Operating Procedure Shop Folder Contents

Rev: 1.0
Cadence: Per casting
Owner: ____________
Standard contents of the physical shop folder — what goes on the cover and what is filed at each of the four tabs, left- and right-hand side. Every folder is built the same way so anyone on the floor can find a document without hunting.

1 Cover

  • 1Cover Sheet — printed and applied to the front of the folder.
  • 2Sticker — printed and applied.

2 Tab-by-tab contents

1 1st Tab
Left hand / inside cover
  • Shop Drawings / Tickets All dimensional information about the concrete components.
Right hand
  • Tracking Sheet
  • Stickers — stapled behind the Tracking Sheet.
2 2nd Tab
Left hand
  • Formwork 3D Drawings
  • Jig List
  • Form Out Instructions If needed.
Right hand
  • Sales Order
  • Form Out Instructions If needed.
3 3rd Tab
Left hand
  • Casting Layout
  • Casting Instructions
Right hand
  • Batching Ticket
  • Color Log
4 4th Tab
Left hand
  • Build Crate Ticket
Right hand
  • Optimizer Hours
Bottom line: cover sheet + sticker on the front, then four tabs — drawings & tracking, formwork & sales order, casting & batching, crate & optimizer — always filed left/right the same way. SOP 9.1 — Shop Folder Contents
