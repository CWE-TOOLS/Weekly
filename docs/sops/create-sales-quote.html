4.1

Standard Operating Procedure Create Sales Quote

Rev: 1.0
Cadence: One per casting
Owner: ____________
Create a Sales Quote for each casting in a project. Every quote must include all material from Mill through Shipping — nothing left off.
1

Open Inflow.

2

Click the + icon, then click Sales Quote.

3

For Customer, select the project you are creating the quote for.

4

Name the quote with the project name and Cast # — ex. St Hilda's Cast #5

5

Add all material required for this casting.

Wood, caulk, casting ingredients, sealers, acid, crating materials, etc. — everything from Mill through Shipping.

If batching has been created for a casting, the material totals are listed above the batch tickets in the Project Portal.

6

Set the Ordered on date to the first of the month the casting starts.

This can be shifted later when adjusting the quote for an end-of-month material order report.

7

Click Save — the Sales Quote is complete.

Bottom line: One quote per casting in Inflow, customer = the project, named "Project + Cast #", every material from Mill through Shipping, "Ordered on" = the 1st of the start month, then Save. SOP 4.1 — Create Sales Quote
