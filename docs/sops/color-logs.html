The Color Log is the recipe of record for a job's concrete mix — cement, sand, aggregates, pigments, additives, grout, and fill coat, exactly as the Concreteworks East paper form. Everyone who batches, finishes, or seals reads from it, and Batch Tickets use it to scale every ingredient to each casting. It lives on the Color Log tab of each project.
Batch Tickets depend on the Color Log. The log must be complete and include at least one Sand entry — if Batch Tickets won't generate, that's the first thing to check.

1 What it's for

One accurate mix recipe per job, shared by the whole floor. It feeds the Batch Tickets tab (which scales ingredients per casting) and is a Project Manager milestone on the Releasability Board — the "Color Log" column tracks whether the color/finish is documented and confirmed.

2 Opening the Color Log

Open a project in the Project Portal, then click the Color Log tab in the left sidebar (palette icon) — it sits between Staging and Batch Tickets. The form opens with the title COLOR LOG — Concreteworks East — Rev. 2025.

New project? Save it first — until then you'll see "Save the project first to use Color Log."

3 The form, section by section

Header
Sample # (color name), Temperature, Project (auto-filled), Date, Made By, a STANDARD / CUSTOM toggle, and Cement Type (White Portland / Gray Portland / Other).
Base Ingredients & Additives
Base Ingredients (left): name, weight, unit, note — must include Sand.   Additives (right): name, amount, unit, note.
Cast Method
Spray Up / Direct Cast / Other — affects how Batch Tickets handle materials.
Aggregates & Pigments
Aggregates (left): name, amount, unit.   Pigments (right): name, %, weight, unit.
Grout Type & Fill Coat
Grout Type (left): name, ratio %.   Fill Coat (right): name, %, weight, unit.
Notes
Finishing Notes and Sealing Notes — free-text instructions for the floor.

4 Filling one out

1

Open the Color Log tab for the project.

2

Optional: use Load Standard Color to start from a saved preset, then adjust it for this job.

3

Fill the header — Sample #, Temperature, Date, Made By. (Project auto-fills.)

4

Set the STANDARD / CUSTOM toggle and pick a Cement Type.

5

Add Base Ingredients with + Add Ingredient — include at least one Sand entry (required for Batch Tickets).

6

Add Additives with + Add Additive.

7

Choose the Cast Method (Spray Up / Direct Cast / Other).

8

Fill Aggregates, Pigments, Grout Type, and Fill Coat using each section's + Add button. Click any field to edit inline; click the row's × to delete it.

9

Write any Finishing Notes and Sealing Notes.

10

Wait for the header to show Saved (it auto-saves as you type — there's no Save button), then Print Color Log if you need a copy for the folder or floor.

5 Presets, multiple logs & the ⋯ menu

Load Standard Color
Drop a saved recipe in as a starting point, then tweak it for the job.
⋯ › Save as Standard Color
Save the current log as a reusable preset (you name it). It then appears in Load Standard Color for any project.
Multiple color logs
Toggle ON for jobs with more than one mix — a pill strip appears with + Add color log and Delete. One-way switch: can't be turned back off.
Notes box
Project-level notes for anything that doesn't fit the form fields.
Print Color Log
Prints the form as shown. Save first so the printout is current.

6 Saving & storage

Everything auto-saves to the database as you type and persists between sessions — watch the Saved indicator near the top-right of the project header. Color logs are per-project; standard-color presets are stored separately and reusable across all projects. Folder labels for castings also include the Color Log at the bottom.

Bottom line: The Color Log is the job's mix recipe. Fill every section (Sand is required), let it auto-save, and print it for the floor. Batch Tickets and the Releasability "Color Log" milestone both depend on it. Color Logs
