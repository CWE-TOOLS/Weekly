Standard Operating Procedure Casting Layout

Rev: 1.0
Cadence: Per casting — before cast day
Owner: ____________
Lay out a casting's forms on the shop-floor casting tables using the Project Portal's Casting Layout tab. The portal auto-packs every panel onto the tables to scale; you fine-tune by dragging and rotating, then print the diagram so the casting floor sets up the tables exactly as planned.
1

Confirm the casting's panels are entered in Casting Inventory with a width and length on every component. Sizes come from the Tracking sheet. Any panel with a missing or unreadable dimension is skipped from the layout — the tab shows “N panel(s) skipped” if so.

2

Open the project in the Project Portal and go to the Casting Layout tab (left sidebar — the project must be saved first).

3

Pick the casting from the Casting dropdown.

4

Choose where it will be cast in the Casting area dropdown — Casting Area B or Casting Area A. Each area draws its own real tables to scale; a layout is saved per casting, per area.

5

Set the Mold offset (per side, in) (default 4") and Space between forms (in) (default 0"). The offset is added to every edge of the concrete size to get the true mold footprint on the table. The layout re-packs live as you change these.

6

Clear the overflow. Anything that didn't fit sits in the red OVERFLOW strip below the tables — drag each overflow form onto open table space. The status line under the controls counts any molds still off a table.

7

Fine-tune by hand. Drag a form to move it; double-click it — or hover and press R — to rotate 90°. A drop must land fully on one table without overlapping another form; an invalid spot flashes red and snaps back. Your first change saves a manual layout, and every change after that saves automatically — there is no Save button.

8

Check the status line before printing: no skipped panels, no overflow, and the panel count matches the casting. A missing panel on the printed diagram means a missing form on cast day.

9

Click Print Layout and print the diagram. Post it with the casting paperwork so the floor builds the tables to the plan.

10

Need to start over? Click Reset to auto-pack (appears once a layout is manual) and confirm — the manual layout for that casting area is discarded and the portal re-packs automatically.

Bottom line: every casting goes to the floor with a printed, to-scale table layout showing where each form sits. Casting Layout
