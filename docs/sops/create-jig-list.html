Standard Operating Procedure Create Jig List

Rev: 1.0
Cadence: Per casting — before release
Owner: ____________
Create a set of drawings describing and listing all screed jigs required to complete a casting. This is one of the items on the Releasability Board for each project folder.
1

Review the concrete components in the project folder and identify:

  • The direction each form will be screeded (typically from the shortest side)
  • The depths and quantity of scrim layers in the concrete (ex: 1 at 1/4" from face of concrete and 1 at 1/2" from face of concrete)
  • All unique jig widths for that casting

A jig can be used across multiple forms if the forms have the same width or are within 1" of each other.

2

Make a list of all jig widths needed for the casting, including one for each scrim level of each width.

3

You will generally want at least two of each unique jig type.

4

Open the Jig List Tool at S:\!CWE - Tools\Custom Tools\Jig Making.

5

Fill out the Project / Casting Information at the top of the page, including the date created.

6

Fill out and confirm the jig depths required, based on the scrim depths identified in step 1.

7

Fill out the Panels section with each unique width from the list in step 2.

8

Name each jig. (ex: by form type or width)

9

Fill out the quantity per jig type. (typically 2)

10

Click Print / Save PDF when all information is complete.

11

Save to the 3D Drawing folder of the project, labeled Jig List – Cast #X.

12

Print, hole punch, and place in the casting folder on top of the 3D drawings for that casting.

13

Also click Export JSON and save under a folder with the project title at S:\!CWE - Tools\Custom Tools\Jig Making\Projects.

You can load this file in the future to edit or make new casting lists.

Bottom line: every casting gets a printed Jig List in its folder and a saved JSON in the Jig Making projects folder before release. Create Jig List
