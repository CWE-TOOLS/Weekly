Standard Operating Procedure Generate Weekly Project Sales Orders

Rev: 1.0
Cadence: Weekly
Owner: ____________
For the upcoming week, create an Inflow Sales Order for every casting on the Phase Start report, covering all material for that week's work and dated the following Monday.
1

Open the Weekly Schedule view.

2

Verify all tasks in the upcoming week are scheduled and complete.

3

Once the week is verified complete, click the printer icon in the top-right corner.

4

Select Phase Start Report and select all departments.

5

Print the Phase Start report — reference it for the following steps.

6

Open Inflow.

7

Hover over the Sales icon on the left sidebar and click Sales Orders.

8

For each casting on the right side of the Phase Start report, create a Sales Order.

9

For the Customer, select the project that casting is in.

10

Title the Sales Order with the casting name and the departments — ex. St Hilda's Cast #1 (Mill, FO, Cast)

11

Add all materials required for that casting's work to be performed the following week. Refer to the casting's existing sales orders / quotes if they exist.

12

Once complete, ensure all castings on the Phase Start report are present, and date them the Monday of the following week.

13

Move on to the Generate Pic List SOP.

Bottom line: Verify the week, print the Phase Start report, then create one Inflow Sales Order per casting (customer = project, named "Casting + departments") with all material, dated next Monday — every casting on the report accounted for. Weekly Project Sales Orders 4.3
