Standard Operating Procedure Generate Batch Tickets

Rev: 1.0
Cadence: Per casting — before cast day
Owner: ____________
Generate and print the concrete batching tickets for a casting from the Project Portal. The generator scales the Color Log recipe to the casting's volume and prints one color-coded page per batch — FACE, FIRST BACK UP, and FINAL BACK UP — for the casting floor.
1

Confirm the Color Log is ready. The project must be saved, a Color Log must exist for the project, and it must include a Sand entry. Tickets scale their ingredients from the Color Log — no log with sand, no ticket. See SOP: Color Logs

2

Verify every component in the casting has its Cu Ft and finish face square footage filled out in Casting Inventory. If any row is missing either value, Import from Tracking won't import the casting's info at all.

3

Open the project in the Project Portal and go to the Batch Tickets tab (left sidebar, just after Color Log).

4

Confirm the project-wide Target Slump for each batch type. Defaults: Face Mix 6", First Back Up 5", FINAL Back Up 5". These draw the slump bullseye on every printed ticket.

5

Pick the casting from the casting pills. On phased projects, batch tickets are handled per phase — work within the phase you're in.

6

Enter the volume: Total Cu Ft (required — the whole recipe scales from this) and Face Sq Ft, or click Import from Tracking to pull both in automatically.

7

Set the pigment reduction % (default 50%) and choose whether the FIRST BACK UP and FINAL BACK UP batches get the reduction.

8

Fill out Batched by and any Notes — both print on the ticket.

9

Review the scaled batches the generator produced — batch count, batch type, and scaled ingredients. The generator splits batches by standard sand sizes, labels them by the Color Log's cast method (Spray Up / Direct Cast), scales the recipe by sand weight, and applies the material rules (sand swaps, fiber→Cemfill, pigment reduction). Full rules — see SOP: Principles of Batching

10

Wait for the save indicator to read “Saved” before printing, so the tickets reflect your latest inputs.

11

Click Print Batch Tickets, pick the castings to print in the dialog, and print. One page per batch, color-coded: FACE orange, FIRST BACK UP blue, FINAL BACK UP yellow.

Bottom line: every casting goes to the floor with a printed, color-coded ticket for each batch, scaled from the Color Log recipe. Generate Batch Tickets
