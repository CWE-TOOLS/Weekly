Standard Operating Procedure Updating Project File Racks

Rev: 1.0
Cadence: Weekly — after Friday Buy-In
Owner: ____________
Applies to the folder racks under the televisions in the production office.
Keep the wall board, file dry boards, and file racks below the TV in sync with the current production week. Cross-check every list against the Optimizer / Releasability spreadsheet before posting.
1

Update the wall dry board

Update the dry board on the wall with the current week # and the dates of the week.

2

Label the file dry boards

Label each file dry board with the week # and a list of all projects that have fabrication starting that week.

Cross-check the Optimizer / Releasability spreadsheet to confirm the list is accurate.

3

Flag not-released projects ★

Any project with a not-released item — internal or external — gets marked with a RED star ★ next to its name.

4

Shift the files

All files are “shifted” / moved after the Buy-In meeting — every Friday.

5

Hold red-sticker buy-in files

Any buy-in files carrying RED stickers (not-released items) stay in the office file rack and are placed on the desk — so the outstanding items can be completed.

6

Re-rack after the plan meeting

After the 1st or 2nd half plan meeting, place files in the file racks below the TV and update the dry boards.

Bottom line: Boards reflect the week, red star = not released, files shift every Friday, red-sticker files stay on the desk until cleared, then re-rack below the TV. SOP 1.3 — Updating Project File Racks
